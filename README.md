# kwater
기본적으로 train 데이터를 가지고 test 데이터를 예측하는데 제공되는 영상을 참고해서 얼마나 작동하는지 확인해야할것같음

대회에서 평가지표가 Macro F1 Score 가 사용된다는데 찾아보니 정밀도 와 재현율 조화 평균이라고함

본 대회의 평가에는 Macro F1 Score가 사용됩니다.
mAP가 아니라 F1이므로 참가자분들은 불필요한 중복 예측이 없도록 Non-Max Suppression 혹은 그에 준하는 정리 과정을 적용한 결과물을 제출하셔야 합니다.

즉 영상에서 물고기가 나타나는 떄 이 물고기가 무엇인지 예측하는것이 중요하고

TP (True Positives), FP (False Positives), FN (False Negatives)는 이진 분류 모델 또는 다중 클래스 분류 모델의 성능을 측정하고 평가하는 데 사용되는 중요한 용어입니다. 이 용어들은 모델의 예측과 실제 라벨 간의 일치 여부에 대한 정보를 제공합니다:

TP (True Positives - 진양성):

모델이 올바르게 예측하고 실제 라벨과 일치하는 경우를 나타냅니다. 모델이 양성 클래스를 양성으로 예측하고, 실제로도 양성 클래스인 경우입니다.
FP (False Positives - 가양성):

모델이 잘못된 예측을 하고 양성 클래스로 예측하면서 실제로는 음성 클래스인 경우를 나타냅니다. 모델이 양성으로 예측하지만 실제로는 음성인 경우입니다.
FN (False Negatives - 가음성):

모델이 잘못된 예측을 하고 음성 클래스로 예측하면서 실제로는 양성 클래스인 경우를 나타냅니다. 모델이 음성으로 예측하지만 실제로는 양성인 경우입니다.
이러한 용어는 다양한 성능 메트릭을 계산하는 데 사용됩니다. 예를 들어, 정밀도(Precision), 재현율(Recall), F1 스코어 등을 계산할 때 TP, FP, FN 값을 사용합니다. 이 메트릭들은 분류 모델의 성능을 평가하고 모델이 얼마나 정확하게 예측하는지에 대한 정보를 제공합니다.

정밀도 (Precision): TP / (TP + FP)
재현율 (Recall): TP / (TP + FN)
F1 스코어 (F1 Score): 2 * (정밀도 * 재현율) / (정밀도 + 재현율)
이 메트릭들을 사용하여 모델의 성능을 종합적으로 평가할 수 있습니다.
